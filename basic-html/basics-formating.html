Headers & Paragraphs

Headers and Paragraphs are the simplest HTML elements, used to create a basic titles and text boxes.

<h1> / <h6> - Header Elements

Headers help organize a website and create a clear structure for readers.

<p> - Paragraph Element

Paragraphs are used to display the main text content of a website.

Anchors

Anchors are a used to move you to a different point on a web page, link different web pages together or even move you to a different website.

<a> - Anchor Element

Anchors use attributes such as href=" " and target=" ".

Line Breaks

Line breaks are used to create separation between elements on a web page by placing them on a new line.

<br> - Break Element

Creates separation with no visual que, mainly used in paragraph elements.

<hr> - Horizontal Rule Element

Creates separation by leaving a visual line across the web page, mainly used to separate different sections of elements.
