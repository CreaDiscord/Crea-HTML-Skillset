Headers & Paragraphs

Headers

Headers help organize a website and create a clear structure.
Headers in HTML range from h1 to h6 tags.

Paragraphs

Paragraphs are used to display the main text/content of a website.
Paragraphs in HTML use p tags.

Anchors

Anchors are used to link pages together or even move from one are of a page to another.
Anchors in HTML use the a tag, along side the attributes href and target.

Line Breaks

hr and br are both used to create separation between elements on a web page and place them on a new line.
hr adds a horizontal line across the page.
br creates a new line in a paragraph.
