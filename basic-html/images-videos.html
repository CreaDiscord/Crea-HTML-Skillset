Images

Images can be loaded onto a web page for multiple purposes.

<img> - Image Element

Images should contain an attribute that contains the URL / file location.

Image Attributes

  • src=" "
  • alt=" "
  • width=" "
  • height=" "

Figures

Figures are used to provide addition information or context to an image or video.

<figure> - Figure Element

Figure is the root element of a figure, nested between are an image / figcaption element.

<figcaption> - Figcaption Element

Figcaption is used to add a caption bellow / above an image in a figure element.

Videos

Videos are used to display videos on a website and should be a pair of nesting tags.

<video> - Video Element

Unlike images, videos do not hold the video's URL / file location.

Video Attributes

  • poster=" "
  • width=" "
  • height=" "
  • controls
  • autoplay
  • loop
  • muted

<source> - Source Element

The source element holds the video URL / file location.

  • src=" "
  • type="video/ "

Audio

Audio is used to play music and other sound effects on a website. They work similarly to video tags but are slightly different.
They are made with the audio tag. Like video, they have a few source tags nested inside holding a type="audio/***".
Audio also takes the controls, loop, autoplay and muted attributes.

iFrames

iFrames are used to embed other webpages and external content.
Created using the iframe tags they take the src, width & height attributes along side new frameborder and allowfullscreen.
