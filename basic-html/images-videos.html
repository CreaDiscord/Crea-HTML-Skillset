Images

Images can be loaded onto a web page for multiple purposes.

<img> - Image Element

Images should contain an attribute that contains the URL / file location.

Image Attributes

  • src=" "
  • alt=" "
  • width=" "
  • height=" "

Figures

Figures are used to provide addition information or context to an image or video.

<figure> - Figure Element

Figure is the root element of a figure, nested between are an image / figcaption element.

<figcaption> - Figcaption Element

Figcaption is used to add a caption bellow / above an image in a figure element.

Videos

Videos are used to display videos on a website and should be a pair of nesting tags.

<video> - Video Element

The video element does not hold the video's URL / file location, instead they go in the source element.

Video Attributes

  • poster=" "
  • width=" "
  • height=" "
  • controls
  • autoplay
  • loop
  • muted

<source> - Source Element

The source element holds the video URL / file location.

Source Attributes

  • src=" "
  • type="video/ "

Audio

Audio is used to play music and other sound effects on a website. They work similarly to video elements.

<audio> - Audio Element

The audio element does not hold the video's URL / file location, instead they go in the source element.

Audio Attributes

  • controls
  • autoplay
  • loop
  • muted

<source> - Source Element

The source element holds the audio URL / file location.

Source Attributes

  • src=" "
  • type="audio/ "

iFrames

iFrames are used to embed other webpages and external content.

<iframe> - iFrame Element

iFrames have many uses, youtube videos and websites are just a couple examples.


iFrame Attributes

  • src=" "
  • width=" "
  • height=" "
  • frameborder=" "
  • allowfullscreen
